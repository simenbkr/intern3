Calendar Demo

This page demonstrates the use of a calendar control built in Angular JS. It allows the user to choose a date. The calendar itself is an Angular JS directive, and I'm using LESS CSS for some of the more complicated CSS rules. I'm also make use of Moment.js, a library that allows for easier manipulation of time than the standard JavaScript libraries. I highly recommend you check them out.
